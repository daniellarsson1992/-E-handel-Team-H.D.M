## Sammanfattning

Vi har arbetat med att analysera försäljningsdata och sammanställt detta i en Jupyter Notebook. Datan som vi använt kommer från en `.csv`-fil.

---

## Instruktioner

För att kunna se resultatet av rapporten behöver man en IDE, till exempel Visual Studio Code.

En del andra programpaket behövs för att kunna köra rapporten. Dessa installeras lättast via den inkluderade `requirements.txt`-filen. Denna fil försäkrar att du använder rätt versioner av alla paket för att undvika kompatibilitetsproblem.

För att installera dessa, öppna kommandotolken/terminalen i projektmappen och skriv in:

```bash
pip install -r requirements.txt
```
